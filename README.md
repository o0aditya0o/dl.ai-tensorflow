# dl.ai-tensorflow

## Setup Instructions

To resolve the `externally-managed-environment` error and start the Python interpreter, follow these steps:

1.  **Activate the Virtual Environment**:
    ```bash
    source .venv/bin/activate
    ```
2.  **Verify Setup**:
    ```bash
    python --version
    pip list
    ```
3.  **Jupyter Notebook**:
    If using these notebooks in VS Code or Jupyter, select the kernel named **"Python (dl.ai-tensorflow)"**.
